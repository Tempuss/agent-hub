# 사례 1: 신규 시장 진출 전략 (Market Entry Strategy)

**상황**: B2B SaaS 회사가 신규 지역으로 확장할지 결정해야 함

---

## Phase 1: 문제 정의

### 상황
```
우리는 미국 시장에서 성공한 HR 관리 SaaS를 운영 중
고객들이 "아시아 태평양 지역으로 확장해달라"고 요청 중
자금 남은 상태, 팀 규모 약 20명

질문: 일본 시장에 진출해야 하는가? 언제 진출해야 하는가?
```

### 성공 기준
```
명확한 GO/NO-GO 결정
리스크 평가
실행 계획 (GO 결정 시)

목표: 신뢰도 70% 이상의 의사결정
기간: 2주 내 의사결정
```

## Phase 2: 사고법 선택 및 적용

### 선택 이유
```
문제 유형: 전략적 의사결정 + 시장 분석
선택한 사고법: SWOT Analysis + GAP Analysis + Kepner-Tregoe
선택 이유:
- 우리 자체의 강점/약점 파악 필요
- 시장의 기회/위협 분석 필요
- 현재 → 목표 상태의 갭 분석 필요
- 복수의 선택지(일본 vs 싱가포르 vs 호주) 중 선택
```

### SWOT 분석

#### 강점 (Strengths)
```
1. 미국 시장에서 검증된 제품 및 비즈니스 모델
   - 월 활성 사용자 5,000명+
   - 고객 만족도 4.5/5.0

2. 기술 탁월성
   - 엔터프라이즈급 보안
   - 다국어 지원 기반 구축됨

3. 자금 여력
   - 6개월 추가 운영 자금 (일본 진출 커버 가능)
```

#### 약점 (Weaknesses)
```
1. 아시아 시장 경험 부재
   - 현지 비즈니스 관행 미숙
   - 규제 환경 이해 부족

2. 소규모 팀
   - 현지 팀 구성 필요
   - 영어-일본어 지원 제약

3. 현지 경쟁 미지수
   - 일본의 로컬 플레이어 분석 부재
```

#### 기회 (Opportunities)
```
1. 시장 성장성
   - 일본 HR 기술 시장: 년 15% 성장률
   - 일본 기업의 디지털 전환 가속화

2. 첫 움직이는 이점
   - 아시아 본격 서비스 제공 기업 부재
   - 빠른 진출 시 시장 리더십 기회

3. 글로벌 확장 발판
   - 일본 성공 → 한국, 싱가포르로 확대 가능
```

#### 위협 (Threats)
```
1. 강력한 로컬 경쟁사
   - Workday, SuccessFactors가 이미 일본 시장 진입
   - 현지 기업들(ADP Japan 등)의 강한 입지

2. 규제 위험
   - 개인정보보호법 (APPI) 복잡도
   - 노동법 준수 복잡성

3. 진입 비용
   - 현지화 개발: $500K+
   - 현지 팀 구성: 월 $50K+
   - 진입 시간: 6-12개월
```

### GAP Analysis

#### 현재 상태
```
시장 진출: 미국만
지역 팀: 없음
현지 언어 지원: 영어만
규제 준수: 미국 GDPR/CCPA만
시장 인지도: 일본에서 0
```

#### 목표 상태 (6개월 후)
```
시장 진출: 미국 + 일본
지역 팀: 일본 현지 팀 3-5명
현지 언어 지원: 영어 + 일본어
규제 준수: APPI, 일본 노동법
시장 인지도: 일본에서 200+ 가입자
```

#### GAP 및 액션
```
1. 현지 팀 구성 갭
   - 액션: 채용 (영업 1명, CS 1명, PM 0.5명)
   - 시간: 2개월
   - 비용: $100K+

2. 제품 현지화 갭
   - 액션: UI/내용 일본어 번역, 법규 준수 기능 추가
   - 시간: 3개월
   - 비용: $150K+

3. 시장 진입 갭
   - 액션: 파트너사 발굴, 마케팅 캠페인
   - 시간: 2개월
   - 비용: $50K+

4. 규제 준수 갭
   - 액션: 법무 검토, APPI 준수 감시 체계 구축
   - 시간: 1개월
   - 비용: $30K (외부 법무)
```

### Kepner-Tregoe: 선택지 비교

#### 고려 중인 선택지
```
Option A: 일본 진출 (지금 시작)
Option B: 싱가포르 진출 (더 작은 시장, 규제 간단)
Option C: 호주 진출 (영어권, 진입 용이)
Option D: 대기 (추가 자금 모금 후)
```

#### 평가 기준
```
시장 규모: 어느 정도 큰가? (중요도: 높음)
진입 용이도: 얼마나 쉬운가? (중요도: 중간)
리스크: 얼마나 위험한가? (중요도: 높음)
ROI: 수익성은? (중요도: 높음)
실행 가능성: 우리가 할 수 있나? (중요도: 높음)
```

#### 평가 결과 (1-10점)
```
                일본    싱가포르   호주    대기
시장 규모       9        5        4      0
진입 용이도     4        8        9      10
리스크          5        7        8      2
ROI 잠재력      9        6        5      0
실행 가능성     6        8        9      10

점수 합계       33       34       35     22
(가중치 적용)

평가: 호주 > 싱가포르 > 일본
하지만 ROI와 장기 전략을 고려하면 일본이 최적
```

## Phase 3: 연구 계획

### 확인해야 할 가정
```
1. 일본 시장 규모 (년 15% 성장이 맞나?)
2. 로컬 경쟁 강도 (얼마나 경쟁 심한가?)
3. 규제 복잡성 (APPI 준수가 실제로 어려운가?)
4. 고객 수요 (일본 기업들이 정말 원하나?)
5. 진입 비용 현실성 ($500K로 충분한가?)
```

### 연구 계획
```
사용 가능한 시간: 3주

연구 대상:
1. 일본 HR 시장 규모 및 성장 → Tier 1: 시장 리포트
2. 경쟁사 분석 → Tier 2: 경쟁사 웹사이트 + 분석 기사
3. APPI 규제 요구사항 → Tier 1: 정부 문서
4. 진입 비용 벤치마크 → Tier 2: 다른 회사 사례
5. 고객 수요 검증 → Tier 3: 직접 인터뷰 (5-10명)
```

## Phase 4: 연구 실행 및 기록

### 연구 1: 일본 HR 기술 시장 규모

```
질문: 일본의 HR 기술 시장 규모는 얼마나 되나?
출처: Gartner Market Research 2024 보고서
출처 신뢰도: Tier 1 (전문 분석 기관)

찾은 정보:
- 2023년 일본 HR 기술 시장: $4.2B
- 예상 성장률: 년 12.5% (CAGR 2023-2028)
- 상위 세그먼트:
  * 인사 관리 (HRIS): 35%
  * 급여 관리: 30%
  * 채용 관리: 20%
  * 기타: 15%

신뢰도 평가: 높음 (Gartner는 업계 표준 조사 기관)
```

### 연구 2: 경쟁 상황

```
질문: 일본 시장의 주요 경쟁사는 누구인가?
출처: LinkedIn 회사 페이지 + IT 프레임 분석
출처 신뢰도: Tier 2 (공개 정보 + 분석 기사)

찾은 정보:
주요 플레이어:
1. Workday (미국): 일본에서 800+ 고객
2. SuccessFactors (SAP): 600+ 고객
3. ADP Japan (로컬): 1,000+ 고객
4. Recruit HR Tech (로컬 신생): 50+ 고객

우리의 입지: 이들 모두와 직접 경쟁 (가능)

신뢰도 평가: 중간 (공개 정보이지만 완전하지 않음)
```

### 연구 3: APPI (개인정보보호법) 요구사항

```
질문: APPI 준수에 어떤 비용이 드나?
출처: 일본 개인정보보호위원회 공식 가이드 + PPC Law 사무소
출처 신뢰도: Tier 1 (공식 규제 기관) + Tier 2 (법률 사무소)

찾은 정보:
- APPI는 GDPR보다 덜 엄격 (준수 비용 낮음)
- 필수 요구사항:
  * 개인정보 보호 정책
  * 제3자 이전 승인
  * 개인정보 요청 처리 체계
  * 보안 기준 수립

- 추정 비용: $30K-50K (초기), $10K/년 (유지)
- 타사 경험 (Tier 2): 유사 회사들 $25K-40K 투자

신뢰도 평가: 높음 (공식 출처 + 실제 사례)
```

### 연구 4: 고객 수요 검증 (직접 인터뷰)

```
질문: 일본 기업들이 우리 같은 솔루션을 원하는가?
출처: 직접 인터뷰 (8명)
출처 신뢰도: Tier 3 (제한된 표본이지만 직접 피드백)

인터뷰 대상:
- 일본 대형 제조사 HR 담당자 (2명)
- 스타트업 HR 담당자 (3명)
- 컨설팅 회사 (2명)
- 벤처캐피탈 (1명)

주요 피드백:
- 수요 있음: "클라우드 기반 HR 솔루션 찾고 있음" (7/8)
- 가격 민감도: "월 $50-300 수준 예상" (평균 $150/직원)
- 로컬화 요구: "일본어 지원은 필수" (8/8)
- 진입 장벽: "신생 기업보다 기존 업체 선호" (6/8)

신뢰도 평가: 중간 (표본 수 적지만 명확한 신호)
```

### 연구 5: 진입 비용 벤치마크

```
질문: 일본 시장 진입 비용 실제는?
출처: CB Insights 스타트업 사례 분석 + 직접 인터뷰
출처 신뢰도: Tier 2 (공개 사례 + 경험자 피드백)

찾은 정보:
유사 회사의 일본 진출 사례:
- Slack: 초기 $500K (2013)
- Dropbox: 초기 $300K (2015)
- 우리 규모 회사 예상: $300-600K (6-12개월)

구체 비용:
- 제품 현지화: $150K
- 현지 팀 구성 (3개월): $100K
- 마케팅/진입: $80K
- 법무/규제: $30K
- 버퍼: $50K
→ 총: $410K (우리 예상과 일치)

신뢰도 평가: 높음 (다수 사례 일치)
```

## Phase 5: 통합 분석 및 신뢰도 계산

### 통합 결론

```
SWOT 분석 결론:
- 우리의 강점 (기술, 자금)이 기회(시장 성장)와 만남
- 약점 (경험 부족)은 관리 가능 (팀 구성, 파트너십)
- 위협(경쟁, 규제)은 실재하지만 극복 가능

GAP 분석 결론:
- 현재→목표 상태 도달에 $410K, 6-12개월 필요
- 우리의 자금/시간 범위 내에서 실행 가능

연구 증거 결론:
- 시장 규모/성장성 확인됨 (시장 OK)
- 고객 수요 어느 정도 확인 (신호 양호)
- 경쟁 심하지만 우리 틈새 있음 (경쟁 관리 가능)
- 진입 비용 현실적 (비용 OK)

최종 판단:
→ 일본 진출은 전략적으로 타당하고 실행 가능
```

### 신뢰도 계산

```
방법 신뢰도: 85%
  - SWOT, GAP, Kepner-Tregoe 모두 잘 증명된 방법
  - 각 방법이 서로를 보완함

증거 신뢰도:
  - Tier 1: 2개 (시장 리포트, 규제 문서) → 90%
  - Tier 2: 2개 (경쟁사 분석, 벤치마크) → 70%
  - Tier 3: 1개 (고객 인터뷰) → 50%

  평균 = (2×0.9 + 2×0.7 + 1×0.5) / 5
       = (1.8 + 1.4 + 0.5) / 5
       = 3.7 / 5
       = 74%

상황 적합도: 90%
  - 시간 충분 (3주 연구 완료)
  - 자금 충분 ($410K 가능)
  - 전략 명확 (진출 목표 명확)

최종 신뢰도 = 85% × 74% × 90% = 56.7%

→ 신뢰도 57% (중간 수준, 파일럿 규모 추천)
```

## Phase 6: 의사결정 및 액션 플랜

### 최종 의사결정

```
결정: 일본 시장 진출 GO (파일럿 규모)

근거:
1. 시장 기본 조건 양호 (시장 성장, 수요 신호)
2. 경쟁 입지 가능 (상위 3개는 엔터프라이즈 포커스, 우리는 중소기업)
3. 진입 비용 현실적 (6개월 자금 내)

신뢰도: 57% (중간)
→ 파일럿 규모로 시작, 3개월 후 재평가

위험 요소:
- 시장 진입 속도가 예상보다 느릴 수 있음 (규제 등)
- 경쟁사가 우리를 베껴서 따라올 수 있음
- 팀 구성이 예상보다 오래 걸릴 수 있음

위험 완화:
- 파트너사를 통한 빠른 시장 진입 (3개월 내)
- 제품 차별화 강화 (AI 기반 채용 기능 등)
- 현지 채용 전담 담당자 즉시 배정
```

### 액션 플랜

#### Phase 1: 실사 및 준비 (1개월)

```
1. 일본 현지 파트너 발굴
   - 소유자: CEO
   - 액션: 현지 HR 컨설팅사 5곳과 미팅
   - 목표: 진출 파트너 1곳 확보
   - 리소스: 여행비 $5K, 시간 40시간

2. 법무 검토 및 APPI 준수 계획
   - 소유자: COO
   - 액션: 일본 법무사무소 1곳 선정, 30일 검토
   - 목표: APPI 준수 로드맵 수립
   - 리소스: 비용 $15K

3. 현지 팀 채용 계획 수립
   - 소유자: HR
   - 액션: Job board 등록, 채용 프로세스 준비
   - 목표: 후보자 3명 이상 확보
   - 리소스: HR 시간 50시간, 채용비 $5K

4. 제품 현지화 로드맵
   - 소유자: Product
   - 액션: 일본어 UI/번역 범위 정의
   - 목표: 필수 기능 일본어화 계획 수립
   - 리소스: Product 시간 40시간
```

#### Phase 2: 현지화 및 팀 구성 (3개월)

```
1. 제품 일본어 번역 및 로컬화
   - 일정: Month 1-2
   - 비용: $80K
   - 성공 기준: 모든 UI 일본어로 변환

2. APPI 준수 기능 개발
   - 일정: Month 2-3
   - 비용: 개발팀 (기존)
   - 성공 기준: 개인정보 삭제/이전 기능 구현

3. 일본 팀 구성
   - 영업 담당자 1명 (파트타임)
   - CS 담당자 1명 (풀타임)
```
